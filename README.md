## 基于Google Authenticator（谷歌认证器）的身份认证系统



**时间：大三上学期听说很水的网络信息安全课上</br>
地点：每周四早起走8分钟抵达的A区恕园13号楼</br>
起因：源于刚归国的学究型老师阿宝的突发奇想</br>
经过：采用口令&谷歌认证器→“双因子认证”登录</br>
结果：2个人熬了一个晚上完成了简易版认证登录**
